## Project Title:
RL-Based Smart Traffic Signal Optimization using SUMO

---

## Description:
This project implements a Reinforcement Learning (RL)-based system to optimize traffic signal control using the SUMO traffic simulator.  
The system dynamically adjusts traffic signals to reduce congestion, minimize vehicle waiting time, and improve traffic flow efficiency at intersections.

---

## Key Features:
- RL-based traffic signal control (Q-Learning & DQN)
- Integration with SUMO using TraCI
- Dynamic reward system for traffic optimization
- Reduces congestion and waiting time
- Supports visualization and performance analysis

---

## How it Works:
The system simulates traffic using SUMO where an RL agent controls traffic signals at an intersection.  
Based on traffic conditions and rewards, the agent learns an optimal signal switching strategy over multiple training episodes.

---

## Technologies Used:
- Python  
- SUMO (Simulation of Urban Mobility)  
- TraCI  
- NumPy
